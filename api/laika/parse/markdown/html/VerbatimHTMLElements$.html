VerbatimHTMLElements

object VerbatimHTMLElements extends AnyRef

The document tree model for all types of verbatim HTML elements. This includes orphaned start and end tags with no matching second tag. It is up to the renderer to decide whether to ignore these, render them escaped or include them as HTML tags.
