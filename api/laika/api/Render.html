Render

class Render[W] extends AnyRef

API for performing a render operation to various types of output using an existing document tree model.

In cases where a render operation follows a parse operation immediately, it is more convenient to use the transformer.Transform API instead which combines a parse and a render operation directly.

Example for rendering HTML to a file:

 val doc: Document = ...

 Render as HTML from doc toFile "hello.html"
W

the writer API to use which varies depending on the renderer
